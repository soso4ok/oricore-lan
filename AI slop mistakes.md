# Design Research: Anti-AI Human-Made Aesthetic

## Studio Analysis

### basement.studio
- **Hero**: Massive, full-width statement: "We partner with the world's most ambitious..." — no stats, no metrics
- **Navigation**: Ultra-minimal. Just brand + a few links
- **Copy**: Confident, casual, human ("We make cool shit that performs")
- **Cards**: Featured projects with real names, real context — not "100% satisfaction" or "47 clients"
- **Spacing**: Generous whitespace. Nothing cramped. Everything breathes

### quadroom.agency
- **White background**, clean and open
- **Rotating slides** instead of static hero sections
- **Real project images** dominate the page
- **Typography**: Large, clear. No 10px labels
- **Minimal nav**: Logo + links, no CTA button in nav

### rossery.com
- **Portfolio-focused**: The whole page is about showing real work
- **Client logos** as social proof (not generic stats)
- **Testimonials** from named people with real URLs
- **Section headers are large**, readable — nothing microscopic

## Common AI Website "Tells" to Avoid

1. **Fake stats** → "47 clients", "100% satisfaction", "12M+ lines" — unverifiable, generic
2. **10px uppercase labels** → `font-size: 10px; text-transform: uppercase; letter-spacing: 0.2em` — the classic AI label pattern
3. **Buzzword headlines** → "Human-in-the-Loop", "Forward-Deployed Agents", "Cognitive Extraction Protocol"
4. **Bento grid metrics cards** → 2x2 or 3x1 grid of big numbers with tiny labels
5. **"100% / 0 / NDA / 48h"** stat grids → meaningless out of context
6. **Identical hero pattern** → Large headline → subtext → two buttons (primary + outline)
7. **Purple-to-blue gradients** or excessive glassmorphism
8. **Generic copy** that could apply to any company ("We provide the missing context layer")
9. **Overly symmetrical layouts** — everything centered, perfectly balanced

## What to Do Instead (Human Design Patterns)

1. **Asymmetric layouts** — let content breathe unevenly
2. **Larger base text** — 16px minimum body, 18-20px preferred
3. **Real social proof** — named clients, real project names, not numbers
4. **Fewer sections** — say less, mean more
5. **Bold typography as the hero** — let the words be the design
6. **White space as a feature** — not every pixel needs content
7. **One CTA, not two** — confident brands don't hedge with "View Protocol" fallback buttons
8. **Personality in copy** — opinionated, specific, not templated

## Applied Changes for This Build

- WHITE background (#FAFAFA)
- BLACK text (#111)
- GREEN accent (#2FCA54) used sparingly
- NO stats grids
- NO 10px labels
- NO "Human-in-the-Loop" section
- BIGGER animation (full-width)
- BIGGER text sizes throughout
- Fewer sections, more impact
- Copy that sounds like a real studio wrote it
